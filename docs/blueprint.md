# **App Name**: SolarSite Showcase

## Core Features:

- Template Selection: AI suggests suitable templates for solar installation company websites based on the their services.
- Content Generation: Generates content to be used on a solar installation website with a tool that assesses the content requirements.
- Customizable Sections: Allow users to pick-and-choose website components (like photo galleries, quote forms, service lists).
- Hosting: Host the website at a provided url

## Style Guidelines:

- Primary color: Deep sky blue (#007BFF) for trust and professionalism.
- Background color: Very light gray (#F0F8FF) to keep the site light and professional
- Accent color: Forest green (#228B22) for a touch of naturalness.
- Font: 'Inter' (sans-serif) for both headings and body text, offering a modern and neutral look.
- Use clean, minimalist icons to represent services and features, ensuring they align with the solar theme.
- Clean, organized layout with clear calls to action.
- Subtle animations when generating a site, providing smooth feedback without being distracting.
